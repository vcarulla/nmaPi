Automatic Nmap 4 Raspberry PI
==================================

## What

Video [Demo](http://www.youtube.com/watch?v=iubOgQMG2_o)

## Dependencies:

* macchanger
* nmap

## nmaPi.sh

### Running it

Just run:

`./nmaPi.sh`


## nmaPi_term.sh


Is the same as nmaPi.sh, but for PC console, is not as automatic. Useful for testing.

## Running it

### Running it

`./nmaPiterm.sh [option]`

#### Options
                -h - This Help.

                -f - Fast Scan.
                -d - Deep Scan {Default Option}.
                -q - Quiet Scan.
